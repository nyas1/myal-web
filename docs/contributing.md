# 💡 Contribute

App data is synced from the main list repo. How to help depends on what you want to change:

- **App additions, updates, broken links/tags, duplicates, or list improvements:** open an issue or pull request in the main list repo: [nyas1/Material-You-app-list](https://github.com/nyas1/Material-You-app-list) — [Create an issue](https://github.com/nyas1/Material-You-app-list/issues/new/choose) (templates cover the details).

- **This website only** (layout, theme, search, filters, docs site): open an issue or pull request in [nyas1/myal-web](https://github.com/nyas1/myal-web).
